# Swooshz Authenticated Direction B

Selected local design-study package for a future authenticated Swooshz Platform implementation.

## Contents

- `direction-b/`: interactive Precision Workspace prototype.
- `shared.css` and `study.js`: local-only synthetic interaction fixture used by the prototype.
- `comparison.md`: the evaluation and advisory recommendation.
- `evidence/`: desktop and mobile Direction B contact sheets.

## Use

Open `direction-b/index.html` locally. The prototype uses synthetic `.example` data only and performs no network requests or production actions.

This package is a visual reference, not a production port. It does not change authentication, workspace, entitlement, launch, or administration behaviour.
