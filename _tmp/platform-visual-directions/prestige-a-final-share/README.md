# Prestige A final share package

This local-only, untracked folder contains the complete finalised Prestige A visual reference for handoff and review.

## Package structure

- `prototype/` - standalone browser prototype, including its local assets and QA notes.
- `documentation/` - selected-direction design contract for future production implementation.
- `evidence/` - final screenshots, responsive captures, reduced-motion evidence, recordings, contact sheets, and QA results.

## Open the prototype

Open `prototype/index.html` in a browser. The prototype is standalone and does not call Platform APIs or import the production shell.

## Key handoff files

- `prototype/index.html`
- `documentation/prestige-a-design-contract.md`
- `evidence/README.md`
- `evidence/qa-summary.json`
- `evidence/recordings/recordings.json`

This package is not tracked by Git and is not included in the production build. It must remain a visual reference until production integration is separately authorised.
